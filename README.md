# 💻 Simple Interest Calculator Web Application

## 📌 Project Overview

This project is a **Simple Interest Calculator** built using **HTML, CSS, and JavaScript**. It allows users to calculate the total interest and final amount based on principal, interest rate, duration in months, and additional days.

The application provides a clean user interface and handles real-time input validation for better user experience.

---

## 🚀 Features

* Calculate **Simple Interest** based on:

  * Principal Amount
  * Interest Rate
  * Time (Months & Days)
* Automatically converts **extra days into months**
* Displays:

  * Total Interest
  * Total Amount (Principal + Interest)
* Input validation for missing values
* Smooth input navigation using **Enter key**
* Responsive and user-friendly interface

---

## 🛠️ Technologies Used

* **HTML5** – Structure of the application
* **CSS3** – Styling and layout
* **JavaScript (ES6)** – Logic and interactivity

---

## ⚙️ How It Works

1. User enters:

   * Principal amount
   * Interest rate
   * Duration in months and days
2. If days exceed 30:

   * Automatically converted into months
3. JavaScript calculates:

   * Interest using formula
   * Final amount
4. Results are displayed instantly on the screen

---

## 📐 Formula Used

Simple Interest is calculated using:

[
\text{Interest} = P \times R \times T
]

Where:

* **P** = Principal Amount
* **R** = Interest Rate
* **T** = Time (Months + Days converted into months)
